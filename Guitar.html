About the project
The goal of this project was to set up some kind of way for me to practice playing the guitar while keeping rythm. Guitar lessons on YouTube were generally just "Here are the chords you need and here is the strumming pattern. Good luck". This was not particularly useful to me. I needed something where I could follow a long and visually see queues of what to strum and where. So I built this short little python script to help me with that. The code and idea itself is not particularly challenging but this project taught me skills such as learning parts of the tkinter library as well as managing different environments using anaconda.
